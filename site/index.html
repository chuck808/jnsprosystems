    Welcome to AppGatePro

    Professional BMX gate timing and biomechanical analysis at your fingertips

    AppGatePro is a comprehensive UCI-compliant gate timing system that provides lab-grade analytics for BMX riders at a fraction of traditional system costs.

    What's in the Box?

    Your AppGatePro system includes:

    • CoreS3 Main Unit - The brain of your timing system with built-in IMU sensors
    • JNS_Lights - Professional start light system with UCI-compliant sequence
    • JNS_Timing - Breakbeam timer for precise finish line detection
    • Remote Control - Wireless gate triggering from up to 100m
    • Sensor Mounts - Precision brackets for bike attachment
    • USB-C Cables - For charging and data transfer
    • Quick Start Card - QR code for instant documentation access

    Key Features

    UCI Compliant Timing

    Random start delays between 0.1-2.5 seconds, meeting all UCI regulations for official timing.

    Real-Time Analytics

    200Hz IMU data collection providing instant feedback on reaction time, power output, and technique.

    Cloud Connected

    Automatic session uploads to web dashboard for detailed analysis and progress tracking.
